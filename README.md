# Beginner Data Science Projects
Repository that contains a possible solution for some data science exercises for beginners.

## Fake news detection
Type: Classification problem

The goal of this project is create a system to understand if the news are real or fake.

## Loan prediction problem
Type: Classification problem

Identify the customers that are eligible for loan amount.

## Mall customers segmentation
Type: Clustering problem

This data set is created only for the learning purpose of the customer segmentation concepts, also known as market basket analysis.

## Find the causes of the happiness
Type: Data exploration problem

Analuze the dataset and try to find the causes of happiness
